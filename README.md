## How I Think About Agents

An AI agent is not just a prompt with tools. It needs an operating model:

- Mission: the business outcome the agent exists to move.
- Scope: what it can decide, what it must ask, and what it must hand off.
- Tools: APIs, data sources, and automations mapped to each task.
- Memory: what should persist, what should expire, and what should never be stored.
- Evaluation: tests, review checkpoints, failure modes, and escalation rules.
- Adoption: the workflow, UI, documentation, and training that make the system usable.
